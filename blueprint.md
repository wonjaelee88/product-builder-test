# Project Blueprint

## Overview

This project is a simple web application that generates lottery numbers.

## Implemented Features

*   **Lottery Number Generation:** Generates 5 sets of 6 random numbers between 1 and 49.
*   **User Interface:** A simple and clean interface with a button to generate numbers and a section to display the results.
*   **Dark/Light Mode Toggle:** A button to switch between a light and dark theme.
